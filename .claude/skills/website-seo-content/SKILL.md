# Website SEO Content

## When to Use
- updating marketing copy
- improving metadata, sitemap, robots, and discoverability
- adding blog/case-study content with SEO checks

## Workflow
1. inspect target pages and related assets
2. update title/meta/canonical/schema content as needed
3. verify sitemap.xml and robots.txt alignment if URLs changed
4. check internal linking across blog, calculators, and case studies
5. run affected tests or spot checks
