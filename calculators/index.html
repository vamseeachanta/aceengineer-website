About These Calculators

These calculators are provided as educational tools for preliminary engineering estimates. They implement industry-standard methods from recognized codes and standards including:

  • DNV-RP-C203: Fatigue Design of Offshore Steel Structures
  • API RP 2A: Recommended Practice for Planning, Designing and Constructing Fixed Offshore Platforms
  • BS 7608: Code of Practice for Fatigue Design and Assessment of Steel Structures
  • ASME BPVC: Boiler and Pressure Vessel Code
Disclaimer: These calculators are for educational and preliminary assessment purposes only. Final engineering calculations should be performed by qualified engineers using validated software and verified against applicable codes and standards. A&CE accepts no liability for results obtained from these tools.
