Hard, auditable guardrails

The enforcement sits below the model, not in a prompt.

🔒Scoped access. Each channel is bound to specific repositories. Everything out of scope returns 404 — verified, not promised.
📝PR-only changes. No force-push, no history rewrite, no deletes. Every change is a reviewable pull request.
⚖Standards-traceable. Every result maps to a DNV / API / ISO / ASME clause.
📊Audit trail. Every action is logged; fine-grained credentials per scope; client data never mixes across scopes.
