How these reports work

Each page queries the Hugging Face datasets-server API at load time, computes summary statistics in-browser, and states its exact data contract (filter clause, row counts, dataset revision sha). Results are screening-grade and de-identified — vessels are referenced only by vessel_db IDs, and reliability flags travel with every row.
