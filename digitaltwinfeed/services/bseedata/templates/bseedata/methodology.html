Data Usage


The following features are also available upon customer request:
  • Add directional drilling statistical summary
  • Make well directional drilling data available for download
  • Provide all api12 information (includes side tracks for drill-down on each well information)
  • Plot only currently producing 3D well paths for easy assessment of which well bores are currently active
  • Improve well paths by color coding certain sections for better visuals : Water depth section as blue and completion section (if available) as red
  • Add other data tab for .xlsx with all field data for 1 shot download
  • Provide casing or well tubular data in more standardized format
  • Timeline visuals of well construction and completion with all key activities including open hole activities
  • Detailed financial (or economic) analysis along with project NPV (Net present value) or IRR (Internal rate of return)
  • A review of reserves vs. production realized to help shed light into accounting and inconsistent reserves reporting practices among integrated and independent energy entities. very useful for financial analysis and oil and gas well asset analysis
  • Assess field development trends in a given region (all over the world) and choose appropriate development plan for future fields
  • Add more well data offshore (Norway, Canada, Brazil etc.) and onshore regions
  • Automate data refresh from all application regions so the application is abreast with data sources
  • Process data in real-time
  • Identify drill centers by grouping wells within a distance. Utilize Machine learning KNN-Cluster method to identify the well within certain drill center radius
  • Provide GoM assets (pipelines, subsea infrastructures etc.) in a given region based on BSEE data center data. Useful for new field acquisition, exploration, installation operations etc.
  • Correct Rig Days (Assign open_hole logging/ops to Drilling instead of Completions).
