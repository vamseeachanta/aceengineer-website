## Vertical 1: Mid-Market Offshore Engineering Firms

### Profile
- **Size**: 10-50 engineers
- **Revenue**: $5M - $50M
- **Location**: Houston, New Orleans, UK, Norway
- **Software**: OrcaFlex, AQWA, ANSYS

### Pain Points ("Hair on Fire" Problems)

1. **Manual Iteration Hell**
   - Sensitivity studies require 50+ model variations
   - Each variation: 30 min setup + 30 min extraction
   - Engineer burnout on repetitive tasks

2. **Standards Compliance Burden**
   - DNV, API, ABS requirements constantly evolving
   - Maintaining S-N curve libraries is tedious
   - Audit trail documentation time-consuming

3. **Software Expertise Gaps**
   - Can't afford dedicated OrcaFlex scripting expert
   - Rely on manual GUI operations
   - Knowledge concentrated in 1-2 senior engineers

| S-N curve lookup errors | 221 validated curves pre-loaded |

- Build case studies with ROI metrics

- Leverage established reputation
