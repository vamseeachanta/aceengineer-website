What standards do you support?

  • Riser/Pipeline: API RP 2RD, API STD 2RD, DNV-OS-F201
  • Offshore Wind: IEC 61400-3, DNV-ST-0126
  • Pressure Vessels: ASME B31.8, API 579

All 221 S-N curves are validated against published standard values with automated regression testing.
