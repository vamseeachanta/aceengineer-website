Custom Calculators

AceEngineer has developed customized web based calculators as per client requirements. Applications are developed in a user-friendly manner, and provide custom reporting, custom charting and dynamic link for easy portability and presentation.


VM Stress:

This application calculates stress in a specified pipe using tension, bending moment and pressure forces as per API-RP-2RD code and provides parametric charting capabilities.





Probability Calculation

Calculate probability and odds based on individual events. Customized charting is also provided for the application.





Statistical Analysis
